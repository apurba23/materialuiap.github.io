What is Material UI Design

Material UI is a design language which was developed by Google. The code name of material UI design is Quantum paper. It makes the use of grid base layouts, responsive animation transitions, padding and depth effects such as lighting and shadows more liberal. Designer Matias explained that, "unlike real paper, our digital material can expand and reform intelligently. Material has physical surfaces and edges. Seams and shadows provide meaning about what you can touch". Although google sates that their new design language is based on paper and ink.

Overview

Material UI gradually be extended throughout Googleâ€™s array of web and mobile products, providing a consistent experience across all platforms and applications. To incorporate the design language into their website or application, for third party developers Google also release application programming interface. The canonical implementation of material UI for web application user interface is called polymer. It consists of the polymer library, a shim that provides a web components API for browser that do not implement the standard natively, and an element catalog, including the paper element collection that features visual element of the material design.
