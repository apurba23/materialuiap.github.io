User Manual

System - 1

We’ve to just select the component & copy the source code. After that we’ve to paste it on the HTML or CSS file. That’s it
First, we’ve to go http://www.material-ui.com / https://getmdl.io then choose the component from the website and select the design.

Secondly, there have a code for the design. If we needed the hole code we’ve to click this icon in left corner. Then a new window open with the code. Then the HTML portion code have to paste in the HTML file and the CSS portion code paste in the CSS file.[For this no CSS code required]

Thirdly, paste the code in HTML or CSS file.

Finally, When we open it in browser, It looks like this.

System - 2

For this first we’ve needed to go http://materializecss.com/ then we need to write the basic HTML code then we’ve to add - linkrel="stylesheet"href="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.1/css/materialize.min.css" in head for Compiled and minified CSS

And scriptsrc="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.1/js/materialize.min.js" in body for Compiled and minified JavaScript.

Then We’ve to choose the component from the website, select the design and copy the code.

After that we’ve to paste the code in the HTML file.

And it’s done.
